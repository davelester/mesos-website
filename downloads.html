Getting the code via source control

The codebase can also be downloaded from the Apache git repository at: https://git-wip-us.apache.org/repos/asf/mesos.git. A read-only git repository is also mirrored on on Github. (Note: we use Review Board for patches, not pull requests on Github)

Once you have downloaded Mesos, follow the getting started instructions, and take a look at the rest of the Mesos documentation.
